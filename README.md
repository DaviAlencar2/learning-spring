# Learning Spring (Spring Boot)

Projeto do curso “Spring Boot 2025: Do Zero à Amazon AWS e Google Cloud com Java, Docker e Kubernetes” (Udemy – Leandro Costa).

Link do curso: https://www.udemy.com/course/restful-apis-do-0-a-nuvem-com-springboot-e-docker/

## O que você vai ser ensinado
- Spring Boot 3.4 + Java 21: API RESTful, controllers, validação e tratamento de exceções
- Persistência: MySQL, Spring Data JPA/Hibernate e migrations com Flyway
- Boas práticas e padrões: DTO, Builder, Factory, Strategy e Clean Code
- Segurança: Spring Security + JWT, CORS, versionamento de API, Content Negotiation e HATEOAS
- Testes: JUnit 5, Mockito, REST Assured e Testcontainers
- Documentação: OpenAPI/Swagger
- Observabilidade: gestão de logs
- Contêineres e CI/CD: Docker/Docker Compose e GitHub Actions
- Deploy: AWS (RDS, ECR, ECS) e GCP (Cloud SQL, Artifact Registry, Cloud Run, GKE)
- Extras: upload/download de arquivos, Excel/CSV, JasperReports/Apache POI, e-mails (Spring Mail), consumo com React

### [OBS] Irei usar o banco de dados PostgreSQL.
